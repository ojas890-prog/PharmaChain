# 📖 Overview

PharmaChain is a web-based pharmaceutical management platform designed to improve medicine availability, inventory tracking, and supply chain transparency.

The platform connects administrators, pharmacies, and customers through a secure and user-friendly interface, enabling efficient medicine management and real-time inventory visibility.

The project aims to reduce medicine search time, improve inventory management, and provide a scalable digital solution for the pharmaceutical ecosystem.

---

# 🚀 Problem Statement

Traditional medicine distribution systems face several challenges:

- Lack of real-time medicine availability
- Manual inventory management
- Poor supply chain visibility
- Difficulty locating nearby pharmacies
- Inefficient stock management
- Limited transparency across stakeholders

PharmaChain addresses these challenges by providing a centralized digital platform for managing medicines, inventory, and pharmacy operations.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/PharmaChain.git
```

Go to the project folder

```bash
cd PharmaChain
```

Configure Firebase

- Create a Firebase Project
- Enable Authentication
- Enable Realtime Database
- Update `firebase-config.js`

Run the project

```bash
Open index.html
```

or

Use Live Server in VS Code.
